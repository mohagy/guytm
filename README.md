# Guyana Vehicle Management System (VMS)

A static, GitHub-ready web application for managing vehicle registrations, licensing, and traffic offences in compliance with the Guyana Motor Vehicles and Road Traffic Act.

## 🚀 Live Demo
Access the system here: [https://mohagy.github.io/guytm/](https://mohagy.github.io/guytm/)

## ✨ Key Features
- **Admin/Officer Dashboard**: Full registry management and license issuance.
- **Traffic Offence Tracking**: Record violations based on specific sections of the Act.
- **Vehicle Lookup**: Search by license plate to retrieve complete vehicle/owner history.
- **Citizen Portal**: Owners can view their specific records.
- **Persistence**: Data is stored in your browser's `localStorage` (No database server required for hosting).

## 🛠️ How to Run Locally
1. Clone this repository.
2. Open `login.html` in any modern web browser.
3. Login using:
   - **Username**: `admin`
   - **Password**: `123`

## ⚙️ Customization
To change the starting data (default users, zones, etc.), edit the `data.json` file.
